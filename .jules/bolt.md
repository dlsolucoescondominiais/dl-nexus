## 2024-05-19 - FastAPI Sync I/O Anti-pattern
**Learning:** In FastAPI, using `async def` for routes that perform synchronous I/O operations (like `requests.post` or calling synchronous SDK clients without an async wrapper) is an anti-pattern. Because FastAPI runs on a single-threaded event loop, a synchronous call inside an `async def` route will completely halt the event loop until the call finishes, blocking all other incoming requests.
**Action:** When adding dependencies like `httpx` is restricted or when dealing with existing synchronous SDKs/I/O, declare the route with `def` instead of `async def`. FastAPI automatically offloads `def` routes to an external thread pool, preventing them from blocking the main event loop and significantly improving throughput and performance under load.
