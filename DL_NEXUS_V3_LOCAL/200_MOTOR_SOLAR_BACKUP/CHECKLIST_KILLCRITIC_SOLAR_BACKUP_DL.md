# Checklist KILLCRITIC - Solar & Backup DL EcoVolt

## 1. Bloqueios Críticos (Dados Faltantes)
O orçamento NÃO deve prosseguir se faltar:
- [ ] Tensão de atendimento.
- [ ] Tipo de ligação (monofásico, bifásico, trifásico).
- [ ] Consumo mensal base.
- [ ] Autonomia desejada para o backup.
- [ ] Lista detalhada de cargas críticas.
- [ ] Potência das cargas críticas.
- [ ] Local previsto para instalação das baterias.
- [ ] Padrão de entrada.
- [ ] Informação sobre motores/bombas/elevadores na carga crítica.
- [ ] Custo atualizado base (Inversor/Bateria).
- [ ] Validação de compatibilidade do inversor com as baterias.
- [ ] Margem de segurança / técnica (20-30%).
- [ ] Impostos e custos de logística.

## 2. Bloqueios de Promessas Indevidas (Comercial)
A proposta NÃO deve conter as seguintes promessas:
- [ ] "Economia garantida"
- [ ] "Backup total garantido"
- [ ] "Nunca ficará sem energia"
- [ ] "100% autônomo"
- [ ] "Funciona qualquer carga sem necessidade de avaliação"
- [ ] "Preço fechado sem Avaliação Técnica"
- [ ] Termos "grátis", "gratuito", "desconto", "promoção"
- [ ] Termo "visita técnica" (USAR SEMPRE: "Avaliação Técnica")

## 3. Mensagem Obrigatória
Toda proposta gerada deve conter a frase exata:
"Dimensionamento preliminar sujeito à Avaliação Técnica, análise da conta de energia, confirmação das cargas críticas, tensão de atendimento, local de instalação e compatibilidade dos equipamentos."
