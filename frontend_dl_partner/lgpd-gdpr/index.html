Conformidade LGPD & GDPR

Última atualização: 03 de abril de 2026

🛡️ LGPD Compliant 🔒 GDPR Ready

A DL Soluções Condominiais LTDA mantém um compromisso permanente com a proteção de dados pessoais, observando a Lei Geral de Proteção de Dados (Lei 13.709/2018 — LGPD) e as melhores práticas internacionais de privacidade, incluindo o Regulamento Geral sobre a Proteção de Dados (GDPR — UE 2016/679).

1. Nosso Compromisso com a Privacidade

🔐 Privacidade por Design (Privacy by Design)

Nossos sistemas e processos são projetados desde a concepção para proteger dados pessoais. O sistema DL Nexus, nossa plataforma de triagem e gestão de leads, implementa controle de acesso por função e registra logs de auditoria.

📋 Minimização de Dados

Coletamos apenas os dados estritamente necessários para a prestação dos nossos serviços de engenharia condominial. Não coletamos dados sensíveis (biometria, saúde, religião, etc.) dos visitantes do nosso site.

🕐 Limitação de Finalidade e Retenção

Os dados são usados exclusivamente para atendimento técnico e comercial. Após a conclusão do serviço, retemos dados pelo prazo legal obrigatório e os eliminamos de forma segura.

3. Tratamento de Dados em Serviços de CFTV

Como prestadores de serviço de CFTV Forense (DL Guardião™), esclarecemos os papéis e responsabilidades:

📹 Papel da DL Soluções

A DL Soluções atua como Operadora de Dados no serviço de CFTV. Realizamos a instalação, configuração e manutenção dos equipamentos. Não acessamos, armazenamos ou processamos as imagens capturadas.

🏢 Papel do Condomínio/Escola

O contratante (condomínio, escola ou empresa) é o Controlador de Dados das imagens de CFTV. É sua responsabilidade: sinalizar as áreas monitoradas, definir prazos de armazenamento, e responder a eventuais solicitações de titulares sobre suas imagens.

Recomendações para Condomínios com CFTV:

  1. Instalar placas de aviso visíveis nas áreas monitoradas
  2. Não direcionar câmeras para o interior de unidades privativas
  3. Definir prazo máximo de armazenamento (recomendado: 30 dias)
  4. Restringir o acesso às gravações ao síndico e pessoas autorizadas
  5. Manter um registro de quem acessou as gravações e quando
  6. Fornecer gravações a autoridades somente mediante ordem judicial

4. Tratamento de Dados em Controle de Acesso

Para os serviços de controle de acesso (DL Gatekeeper™) que envolvem biometria, RFID ou reconhecimento facial:

  • Dados biométricos são classificados como dados sensíveis pela LGPD (Art. 5º, II)
  • O tratamento requer consentimento específico do titular (Art. 11, I)
  • Recomendamos que o condomínio disponibilize alternativa não-biométrica de acesso
  • Os dados biométricos devem ser armazenados localmente no dispositivo de controle, não em nuvem

5. Segurança da Informação

Medidas técnicas e administrativas implementadas:

  • Transmissão: Criptografia TLS 1.3 em todas as comunicações web
  • Armazenamento: Dados armazenados em servidores com criptografia em repouso
  • Acesso: Princípio do menor privilégio (PoLP) — acesso por necessidade
  • DL Nexus: Plataforma de gestão com autenticação e logs de auditoria
  • Incidentes: Procedimento de resposta a incidentes documentado
  • Treinamento: Equipe técnica orientada sobre proteção de dados

6. Transferência Internacional de Dados

Nosso site utiliza serviços que podem envolver transferência internacional de dados:

  • Google Maps API: para exibição de localização (servidores Google)
  • YouTube: para vídeos institucionais (modo privacidade aprimorada)

Essas transferências são amparadas por cláusulas contratuais padrão (Standard Contractual Clauses — SCCs) e políticas de privacidade dos respectivos provedores.
